| Props | 说明 | 参数类型 | 支持 |
| -- | ---- | ---- | ---- |
| type | 按钮级别 | String | 'primary' / 'secondary' / 'tertiary' / 'warning' / 'danger' 
| icon-name | icon 名字 | String | 
| icon-position | icon 位置 | String | 'left' / 'right'
| loading | 是否为加载按钮 | Boolean | true / false
